ExaEpi
------

ExaEpi is a next generation, agent-based epidemiological modeling code based on `AMReX <https://github.com/AMReX-Codes/amrex>`__.

.. warning::

   ExaEpi is under early development.
   Please contact us with any questions on it or if you like to contribute to its development.


.. _contact:

Contact us
^^^^^^^^^^

The `ExaEpi GitHub repo <https://github.com/AMReX-Codes/ExaEpi>`__ is the main communication platform.

For bug reports or to request new features, you can also open a new `issue <https://github.com/AMReX-Codes/ExaEpi/issues>`__.

We also have a `discussion page <https://github.com/AMReX-Codes/ExaEpi/discussions>`__ on which you can find already answered questions, add new questions, get help with installation procedures, discuss ideas or share comments.
